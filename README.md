### React Notes App

This is a simple app made just to learn Context Api. This app tracks user's notes in a readable form and it can add date, title and 
additional text for the note. Also there is a possibility to delete or add notes.

Technologies Used So Far:
1. React
2. Context Api
3. CSS
4. Bootstrap
4. Git
